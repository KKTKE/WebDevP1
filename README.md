# WebDevP1
BBIT 2.1 Web Dev Project 1

## About
Car Hub is a responsive automotive platform developed for BBIT 2.1 Web Development coursework. Built with HTML5, CSS3, JavaScript, and Bootstrap 5, it allows users to explore new/used cars, compare vehicles, read blogs, and contact the team. The site features a consistent gold and dark theme across eight interconnected pages.

## Features
The platform includes car listings with twelve vehicles (six new, six used) displaying specifications like price, horsepower, and fuel type. An interactive comparison tool lets users select any two cars and view their specs side-by-side. The contact page includes real-time form validation with name (min 2 chars), email (must contain @), and message (min 5 chars) requirements. A blog section features three automotive articles, while the FAQ page uses Bootstrap's accordion for collapsible Q&A. The about page introduces the team members, and the entire site is fully responsive with a cohesive dark theme.
